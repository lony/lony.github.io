Internet Protocol version 6

programming network osi ipv6
IPv6 is the replacement version of the existing IPv4 protocol version for transmitting packages on the internet. Its most visible change is the change of 32-bit addresses to 64-bit, which you can notice having characters on top of the already used numbers. The transition is happening slowly. Looking at a statistic compiled by Google so far only ~25% of the global internet is able to use IPv6.
